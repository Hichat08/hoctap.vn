Ghi nhớ nhanh

Nếu cần học thuộc nhanh, bạn có thể nhớ theo nhóm thay vì học rời rạc từng dòng.

Khung HTML cơ bản

Cấu trúc: <!DOCTYPE html> - <html> - <head> - <body>
Nội dung chữ: h1-h6, p, br, hr, div, span
Đa phương tiện: img, audio, video, source, track, iframe

Form và bảng

Form: form, input, textarea, button, select, option, fieldset, legend, label
Bảng: table, tr, th, td, thead, tbody, tfoot, caption
Danh sách: ul, ol, li, dl, dt, dd

CSS và mạng

CSS nên nhớ: color, background-color, border, font, padding, margin
Mạng: TCP/IP, HTTP, HTTPS, SMTP, POP3, IMAP, FTP, DNS
Wi-Fi: Router, SSID, Mobile hotspot

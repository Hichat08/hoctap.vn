Trang chính

Ghi nhớ nhanh

Nếu cần học thuộc nhanh, bạn có thể nhớ theo nhóm HTML, CSS, mạng và Python thay vì học rời rạc từng dòng.

Luyện trắc nghiệm: Mở bộ câu hỏi ngắn để kiểm tra xem mình còn nhớ được bao nhiêu ý chính.

Mở trắc nghiệm

Khung HTML cơ bản

Cấu trúc: <!DOCTYPE html> - <html> - <head> - <body>
Nội dung chữ: h1-h6, p, br, hr, div, span
Đa phương tiện: img, audio, video, source, track, iframe

Form và bảng

Form: form, input, textarea, button, select, option, fieldset, legend, label
Bảng: table, tr, th, td, thead, tbody, tfoot, caption
Danh sách: ul, ol, li, dl, dt, dd

CSS và mạng

CSS nên nhớ: color, background-color, border, font, padding, margin
Mạng: TCP/IP, HTTP, HTTPS, SMTP, POP3, IMAP, FTP, DNS
Wi-Fi: Router, SSID, Mobile hotspot

Python cần nhớ

Nhập xuất: print(), input(), int(), float()
Điều kiện: if, elif, else và luôn nhớ dấu :
Vòng lặp: for, while, range()
List và hàm: [], append(), len(), def

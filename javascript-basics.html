JavaScript Cơ bản

JavaScript giúp tạo được tương tác trên trang web. Phần này học những cú pháp cơ bản để bạn có thể làm những hành động đơn giản như lấy dữ liệu từ form, hiển thị/ẩn nội dung, và xử lý sự kiện.

Luyện trắc nghiệm: Kiểm tra lại biến, điều kiện, vòng lặp, DOM và sự kiện trước khi chuyển sang bài tiếp theo.

Mở trắc nghiệm

Biến - Variable

var, let, const: Dùng để tạo biến lưu trữ dữ liệu.
let name = "Bạn";
const age = 18;
Tip: Ưu tiên dùng const hoặc let thay vì var.

Kiểu dữ liệu - Data Types

String: let text = "Hello";
Number: let count = 42;
Boolean: let isActive = true;
Array: let fruits = ["táo", "cam"];
Object: let user = {name: "Tuấn"};

Toán tử - Operators

Số học: + - * / %
So sánh: == === != < > <= >=
Logic: && (and) || (or) !(not)
Gán: = += -= *= /=

Điều kiện - If / Else

if (age >= 18) { ... }
else if (age >= 15) { ... }
else { ... }
Ternary: condition ? "yes" : "no"

Vòng lặp - Loops

For: for (let i = 0; i < 5; i++) { }
While: while (x < 10) { }
Do-while: do { } while (x < 10)
For...of: for (let fruit of fruits) { }

Hàm - Functions

Khai báo: function greet(name) { }
Arrow: const add = (a, b) => a + b;
Return: Trả về kết quả từ hàm
function sum(a, b) { return a + b; }
Công dụng Cú pháp Ví dụ
Lấy phần tử HTML document.getElementById() let el = document.getElementById("demo");
Đổi nội dung element.textContent = "..." el.textContent = "Nội dung mới";
Đổi HTML element.innerHTML = "..." el.innerHTML = "<b>Bold</b>";
Gắn sự kiện element.addEventListener() btn.addEventListener("click", function() { });
Thay đổi kiểu CSS element.style.property = "value" el.style.color = "red";
In ra console console.log() console.log("Debug:", value);
Sự kiện - Events Khi nào xảy ra Ví dụ
click Khi người dùng click chuột button.addEventListener("click", handler);
input Khi người dùng gõ vào input input.addEventListener("input", handler);
change Khi giá trị thay đổi select.addEventListener("change", handler);
submit Khi submit form form.addEventListener("submit", handler);
mouseover Khi chuột di vào phần tử el.addEventListener("mouseover", handler);
focus Khi input được focus input.addEventListener("focus", handler);

💾 SQL & Cơ sở dữ liệu

SQL là ngôn ngữ dùng để truy vấn và quản lý dữ liệu trong cơ sở dữ liệu. Phần này học cú pháp cơ bản như SELECT, INSERT, UPDATE, DELETE để bạn có thể làm việc với dữ liệu có tổ chức.

Cơ sở dữ liệu - Database

Bảng (Table): Chứa hàng (rows) và cột (columns)
Bản ghi (Record): Một hàng dữ liệu
Trường (Field): Một cột có tên riêng
Khóa chính (Primary Key): Giá trị duy nhất để nhận dạng bản ghi

SELECT - Lấy dữ liệu

Lấy toàn bộ: SELECT * FROM users;
Lấy cột cụ thể: SELECT name, age FROM users;
Có điều kiện: SELECT * FROM users WHERE age > 18;
Sắp xếp: ORDER BY age DESC;

INSERT - Thêm dữ liệu

INSERT INTO users (name, age) VALUES ('Tuấn', 18);
INSERT INTO users VALUES (1, 'Hoa', 17);
Tip: Khi có khóa tự tăng, có thể bỏ qua ID

UPDATE - Sửa dữ liệu

UPDATE users SET age = 19 WHERE name = 'Tuấn';
UPDATE users SET age = age + 1;
Cảnh báo: Luôn có WHERE để tránh sửa toàn bộ bảng!

DELETE - Xóa dữ liệu

DELETE FROM users WHERE id = 5;
DELETE FROM users WHERE age < 15;
Cảnh báo: Luôn có WHERE, nếu không sẽ xóa toàn bộ!

WHERE - Điều kiện tìm kiếm

So sánh: = != < > <= >=
Logic: AND, OR, NOT
Ví dụ: WHERE age > 15 AND city = 'Hà Nội'
LIKE: WHERE name LIKE 'T%' (tìm tên bắt đầu T)
Lệnh Ý nghĩa Ví dụ
DISTINCT Lấy giá trị duy nhất, không trùng lặp SELECT DISTINCT city FROM users;
COUNT() Đếm số bản ghi SELECT COUNT(*) FROM users;
SUM() Tính tổng cộng SELECT SUM(amount) FROM orders;
AVG() Tính trung bình SELECT AVG(price) FROM products;
MAX() / MIN() Giá trị lớn / nhỏ nhất SELECT MAX(age) FROM users;
GROUP BY Nhóm kết quả theo trường SELECT city, COUNT(*) FROM users GROUP BY city;
ORDER BY Sắp xếp kết quả ORDER BY age ASC; (từ nhỏ đến lớn)
LIMIT Giới hạn số kết quả trả về SELECT * FROM users LIMIT 5;
Bảng ví dụ: users Cột
id (Primary Key) 1, 2, 3, 4, 5
name Tuấn, Hoa, Minh, Lan, Hùng
age 18, 17, 19, 16, 18
city Hà Nội, TP.HCM, Đà Nẵng, Hà Nội, TP.HCM

Ví dụ thực tế

Q1: Lấy tất cả user có tuổi ≥ 18?
SELECT * FROM users WHERE age >= 18;
Q2: Đếm có bao nhiêu user ở Hà Nội?
SELECT COUNT(*) FROM users WHERE city = 'Hà Nội';
Q3: Thêm user mới?
INSERT INTO users (name, age, city) VALUES ('Kiên', 20, 'Hải Phòng');
Q4: Thay đổi tuổi của Tuấn?
UPDATE users SET age = 19 WHERE name = 'Tuấn';

Python Cơ bản

Python là ngôn ngữ lập trình dễ đọc, cú pháp gọn và rất phù hợp để làm quen với tư duy lập trình. Phần này tập trung vào các lệnh nền tảng như nhập xuất dữ liệu, điều kiện, vòng lặp, danh sách và hàm.

Luyện trắc nghiệm: Kiểm tra nhanh cú pháp print(), input(), if, for, while và def.

Mở trắc nghiệm

In và nhập dữ liệu

In ra màn hình: print("Xin chào")
Nhập dữ liệu: name = input("Tên: ")
Lưu ý: input() trả về kiểu chuỗi nếu chưa ép kiểu.
Ép kiểu số: age = int(input("Tuổi: "))

Biến và kiểu dữ liệu

Biến: score = 9.5
Chuỗi: name = "Lan"
Số nguyên: count = 12
Boolean: passed = True
Danh sách: colors = ["đỏ", "xanh"]

Toán tử thường dùng

Số học: + - * / % //
So sánh: == != > < >= <=
Logic: and, or, not
Gán: = += -= *=

Điều kiện if / elif / else

if score >= 8:
elif score >= 5:
else:
Quan trọng: Sau điều kiện phải có dấu : và khối lệnh phải thụt lề.

Vòng lặp for / while

For: for i in range(5):
Duyệt danh sách: for item in colors:
While: while n > 0:
Mẹo: Dùng for khi biết số lần lặp, while khi lặp theo điều kiện.

Danh sách - List

numbers = [1, 2, 3]
Truy cập: numbers[0]
Thêm phần tử: numbers.append(4)
Độ dài: len(numbers)

print("Xin chào", name)
Tham số: Giá trị truyền vào hàm
return: Trả kết quả từ hàm khi cần

Ép kiểu sang số nguyên int(...) age = int(input("Tuổi: "))
Lấy số phần tử hoặc độ dài chuỗi len(...) len("Python")

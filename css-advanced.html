CSS Nâng cao

CSS không chỉ là màu và kích cỡ. Phần này học cách bố trí linh hoạt (Flexbox, Grid), làm đẹp với hiệu ứng, và tối ưu cho điện thoại (responsive).

Luyện trắc nghiệm: Ôn nhanh selector, Flexbox, Grid và responsive bằng bộ câu hỏi tự chấm điểm.

Mở trắc nghiệm

Selectors - Bộ chọn

ID: #myid { }
Class: .myclass { }
Thẻ: p { }
Ghép: div.header { }
Bao: div p { } (p con của div)
Con trực tiếp: div > p { }
Bên cạnh: h1 + p { } (p ngay sau h1)

Pseudo-classes

Hover: a:hover { } (khi rê chuột)
Active: a:active { } (khi click)
Focus: input:focus { } (khi được chọn)
First child: p:first-child { }
Nth child: li:nth-child(2n) { }

Flexbox - Bố trí dòng/cột

Bước 1: display: flex;
Chiều: flex-direction: row | column;
Căn lề trục chính: justify-content: center;
Căn lề trục phụ: align-items: center;
Khoảng cách: gap: 10px;

Grid - Bố trí lưới

Bước 1: display: grid;
Cột: grid-template-columns: 1fr 1fr 1fr;
Hàng: grid-template-rows: 100px 200px;
Lặp: repeat(3, 1fr) = 3 cục bằng nhau
Khoảng: gap: 15px;

Transform & Transition

Dịch: transform: translateX(10px);
Quay: transform: rotate(45deg);
Phóng to: transform: scale(1.2);
Chuyển động: transition: all 0.3s ease;

Hiệu ứng & Độ trong suốt

Bóng: box-shadow: 0 4px 8px rgba(0,0,0,0.2);
Bóng chữ: text-shadow: 2px 2px 4px gray;
Opacity: opacity: 0.5; (50% trong suốt)
Z-index: z-index: 10; (thứ tự xếp chồng)
Tính năng Cú pháp Ý nghĩa
Animation animation: slide 1s infinite; Chạy hiệu ứng @keyframes lặp vô hạn
Gradient background: linear-gradient(to right, red, blue); Tạo nền chuyển màu
Filter filter: blur(5px); Làm mờ hình ảnh
Clip-path clip-path: circle(50%); Cắt phần tử thành hình
Visibility visibility: hidden; Ẩn nhưng vẫn chiếm không gian

Responsive - Thích ứng điện thoại

Media query: @media (max-width: 600px) { }
Mobile first: Viết CSS cho điện thoại trước
Breakpoints thường: 480px, 600px, 768px, 1024px
@media (max-width: 600px) { .grid { grid-template-columns: 1fr; } }

Units - Đơn vị CSS

px (pixel): Cố định, 1px = 1 điểm ảnh
em: Tương đối với font-size cha
rem: Tương đối với font-size gốc
% (phần trăm): Tương đối với cha
vw, vh: Tương đối với màn hình
Ví dụ Flexbox Kết quả
.container {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 100vh;
} 		Xếp phần tử vào giữa màn hình (cả x và y)
.box {
  flex: 1;
} 		Box mở rộng bằng nhau, chia đều không gian
.container {
  display: flex;
  gap: 15px;
} 		Khoảng cách 15px giữa các item

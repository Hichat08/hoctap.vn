Kiến thức mạng cần học

Tổng hợp các khái niệm cơ bản thường gặp trong phần mạng máy tính.

Nội dung Từ khóa Ghi nhớ
Chia sẻ Internet từ điện thoại qua Wi-Fi Mobile hotspot Bật điểm phát Wi-Fi từ điện thoại.
Loại giao thức đặc thù của Internet TCP/IP Bộ giao thức nền tảng của Internet.
Đảm bảo dữ liệu đầy đủ, đúng thứ tự TCP Có cơ chế kiểm tra và sắp xếp gói tin.
Định tuyến gói tin giữa các thiết bị IP Xác định địa chỉ và hướng đi của gói tin.
Truy cập web HTTP Giao thức truyền tải trang web.
Phiên bản bảo mật có mã hóa HTTPS HTTP + SSL/TLS để mã hóa dữ liệu.
Gửi email SMTP Giao thức gửi thư điện tử.
Nhận email POP3 / IMAP Giao thức tải và đồng bộ thư điện tử.
Truyền file giữa máy tính và server FTP File Transfer Protocol.
Chuyển đổi tên miền DNS Đổi tên miền thành địa chỉ IP.
Thiết bị kết nối và định tuyến mạng Router Kết nối các mạng và định tuyến dữ liệu giữa chúng.
Tên hiển thị của mạng Wi-Fi SSID Tên thường thấy khi tìm và kết nối Wi-Fi.
Địa chỉ IP IP address Địa chỉ duy nhất của thiết bị trên mạng.
Subnet mask Subnet mask Xác định phần mạng và phần host của IP.
Cổng mặc định Gateway Điểm ra khỏi mạng nội bộ.
Tường lửa Firewall Bảo vệ mạng khỏi truy cập trái phép.
Mạng cục bộ LAN Mạng trong phạm vi nhỏ như nhà hoặc văn phòng.
Mạng diện rộng WAN Mạng kết nối nhiều LAN trên diện tích lớn.
